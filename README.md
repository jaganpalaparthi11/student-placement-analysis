# Student Placement Analysis

## Overview
This project analyzes student academic and skill data to predict placement outcomes using Logistic Regression.

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Steps
- Data cleaning & EDA
- Feature encoding
- Logistic Regression model
- Evaluation using accuracy, confusion matrix, ROC-AUC

## Result
Achieved ~79% accuracy on test data.
Identified CGPA, internships, and aptitude score as key factors.

## Conclusion
The project demonstrates an end-to-end data science workflow for a real-world classification problem.
